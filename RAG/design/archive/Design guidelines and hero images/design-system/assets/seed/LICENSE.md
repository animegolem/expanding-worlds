# Seed image set — provenance and license

These nine images are PROGRAMMATICALLY GENERATED placeholders
(gradient/pattern compositions produced by `scripts/generate-seed.mjs`
in this repository). They contain no third-party material and are
dedicated to the public domain (CC0 1.0).

The three "artists" they are grouped under — Aster Vale, Juniper
Brandt, Milo Ferren — are fictional names invented for the example;
no attribution to any real artist is made or implied.

This set is a stand-in: it is queued for replacement by a small
curated public-domain art set (owner eyeball on the selection —
that swap is content, not code; the seeding logic in
`apps/desktop/src/utility/seed-library.ts` reads whatever images
live in this directory). Keep the replacement set small (≤ ~12
images, a few hundred KB total) and record each work's source and
public-domain status here when it lands.
