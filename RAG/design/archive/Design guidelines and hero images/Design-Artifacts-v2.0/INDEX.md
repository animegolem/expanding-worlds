# Design Sprint 2 — rollup (Design-Artifacts v2.0 · 2026-07-07)

The complete output of the design sprint, versioned per Design-letter-3's
convention (replaces Design-Artifacts-v1.0.zip in `RAG/design/`).
Compiled against RFC rev 0.54. Every `.dc.html` opens directly in a
browser (keep this folder intact — the documents load `support.js`,
`doc-page.js`, and the `design-system/` examples by relative path).
`Note Lifecycle Document (standalone).html` is fully self-contained
for passing around solo.

## Read in this order

1. **Design-Team-Letter-1.md** — the decision log for the LEAD:
   every ratified call, every `→ RFC` amendment candidate, with
   pointers into the documents below.
2. **Letter-to-the-Designer.md** — orientation for the DESIGNER
   producing the style kit: what's ratified vs open craft, the
   asset production list, how to work here.
3. **Style Guide v2.dc.html** — the visual system as one sheet: prose +
   `■ TECH` build-contract blocks + the live example cards embedded.
   (`design-system/STYLE-GUIDE.md` is the same content in markdown for
   repo consumption.)
3. **Usage Storyboard.dc.html** — the app, function by function, 20
   sections; replaces *How It Works*. Amber chips mark provisional
   positions with their Design-letter-3 item numbers.
4. **UI Vision v2.dc.html** — seven canonical hero states (replaces the
   v1 five-state doc): at rest · open (the bound page) · find · see ·
   collect · the void · the library beside your world.

## Exploration records (the decision trails)

- **Icon Document.dc.html** — glyph styles, the 2006-hybrid turn, the
  merged doctrine mock, tooltip shapes, the tear-off tether, object
  shapes, the open-note gesture. Turns stack newest-first; option ids
  (1a, 2b…) are stable and cited by the letter.
- **Settings Document.dc.html** — the full settings takeover.
- **Gallery Document.dc.html** — the §14.4 controls audit, the search
  track, Space preview, curation facet.
- **Note Editor Wireframes.dc.html** — edit-surface story, rich-text
  controls, broken-link tone rounds, the type rounds that chose
  Maple Mono.
- **Note Lifecycle Document.dc.html** — the FOCUS document for the
  style kit: the three reversible states (open book · sticky ·
  push-pinned landmark) with the → RFC §8.5 amendments, and the
  seven-beat storyboard of one page's day beneath.
- **Menus Document.dc.html** — the context-menu grammar (per-kind
  verb inventories, EPIC-016) + trash browser and ☰/Help tone.
- **First-Run Document.dc.html** — all seven guide pages with
  final-candidate copy (EPIC-019).
