▤ outliner — the invariant grammar

what must stay true about this surface regardless of restyle, density, or platform. the kit (Outliner UI Kit.dc.html) shows the pixels; this page records the rules. grounded in RFC §14.1; badge vocabulary per rev 0.17; every rule below survives desktop and touch alike.

1 · the shape

an org tree master pane + a cursor-follow preview pane. the tree carries structure (fold, alias, loose bin); the preview carries media, prose, and verbs. moving the cursor or tapping a row is never a click cost — the preview always shows the row under attention.

fold·kind glyph·title·badges·#tagsmeta
▾⬚the northern reach#region8

rows are navigation, never inputs. text entry lives in the preview pane and in panels; a tree row never becomes an editable field.

2 · kind glyphs

⌂the root board — heads the outline, never stashed, never trashed
⬚a board — folds, dives, previews as a filmstrip
◯a pin / dot node
▣a bare image node
↗an alias — a canvas already open on the expansion path; activating flies to the real entry, it never unfolds twice
⊘the loose bin — root-level, a workflow home, not an error state

3 · badge vocabulary — three axes, one calm rule

·orphan — no words. the node has no note; it cannot carry wiki-link edges. always visible.
·loose — no place. a location fact, not a connection fact; a loose note may be richly linked. always visible.
·untagged — no taxonomy. the third axis. badges only while a cleanup facet is active — the worklist itches, the browsing tree stays calm.

disconnected = loose ∪ orphan, unchanged. untagged is a facet beside them, never folded into disconnected.

4 · naming fallback — never a raw id

a titled note names its node everywhere. without one:

— images display their source filename in mono, muted. it reads as provenance, not a name (§4.3), and is honest about what the thing is.
— boards read unnamed · 8 items. the real fix is upstream at the naming moment (B1 — name it, enter, you're inside); the outliner just makes un-naming visible enough to want to fix.
— a raw UUID or timestamp-prefix short code never reaches the surface.

5 · filters flatten

the tree renders only at all + empty query. any facet or ⌕ text flattens the result into a worklist whose meta column carries the path — search results never pretend to be a tree. clearing the facet restores the tree with fold state intact.

6 · the preview pane

— follows the cursor / tap selection. never a click cost.
— board rows preview as a filmstrip: first 4–5 children by render_order, thumbnails via the 076 derivative pipeline, non-image children render as their appearance glyph chip — the strip never lies about what's inside. +N closes it. cached LRU on canvasId + revision.
— the empty note area is editable: placeholder "add a note…", commit = create-and-attach in one undoable command. inline capture lives here, never in tree rows.
— verbs adapt: boards lead with ↵ dive in, nodes with ␣ place; fly-to disables at zero placements with its reason; ✎ on a note-less node creates and attaches.

7 · verbs — one inventory, three doors

the preview's verb row, the row context menu (right-click / long-press), and the keyboard reach the same inventory: dive · place · fly to · open note / add a note… · tag… · move to trash. no door offers a verb another door lacks. menus print shortcuts in mono; danger rows use the danger ink.

8 · lens

tapping any tag chip engages the lens: rows carrying the tag keep full strength and ring in the note-orange hit treatment; everything else dims to ~0.35. the lens is view state — it survives fold, filter, and scroll until dropped (esc, or the active-tag chip's ✕ in the bar). one lens, shared grammar with §4.8's tag panel and §7.5 highlight.

9 · the touch dialect

— same skeleton, one density switch: rows grow to ≥44px, verbs become tap chips in the preview. nothing is removed, nothing rearranged.
— portrait: tree full-width, preview rises as a bottom sheet on tap.
— long-press = the context menu door. tap fold glyph folds; tap row previews; double-tap dives; drag a row out places it.
— every keyboard exit has an on-screen twin: ✕ beside esc, chips beside chords.

10 · beats

every verb answers: dive closes the takeover and the camera arrives; alias activation flies with the arrival pulse; place hands off to place mode; attach-note commits one command and says so. esc returns to the board with the camera untouched. silence is a bug.
